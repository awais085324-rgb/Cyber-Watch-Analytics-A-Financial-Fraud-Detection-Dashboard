# Cyber-Watch-Analytics-A-Financial-Fraud-Detection-Dashboard
"The Cyber Watch Portal," provides real-time monitoring and predictive insights into fraudulent transaction patterns. It analyzes 10,000+ transactions to identify high-risk time windows, geographic hotspots, and device-level vulnerabilities.
🛡️ Project Overview
Description: This Power BI dashboard, "The Cyber Watch Portal," provides real-time monitoring and predictive insights into fraudulent transaction patterns. It analyzes 10,000+ transactions to identify high-risk time windows, geographic hotspots, and device-level vulnerabilities.

💡 Key Business Insights
Summarizing the core data logic to demonstrate analytical depth:

🌙 Temporal Risk: 65% of fraud attempts occur during the "Graveyard Shift" (11 PM – 4 AM).

💰 Value Variance: Fraudulent transactions average 15x higher value than legitimate activity.

🔍 Technical Correlation: A strong 89% correlation exists between IP Risk Scores (> 0.85) and confirmed fraud cases.

🌍 Geographic Concentration: 3 Countries account for over 70% of total fraud value.

🛠️ Technical Features
Key skills and tools utilized in this build:

🧬 Data Modeling: Architected a schema linking merchant categories to dynamic risk scores for granular analysis.

🔢 DAX Measures: Developed custom measures for Average Risk Score, Fraud vs. Safe ratios, and total value distributions.

🎨 UI/UX Design: Designed a high-contrast, dark-themed canvas featuring a "Red Alert" status indicator for immediate risk identification.

📊 Visualizations: * Donut Charts for merchant and transaction mode distribution.

Dual-axis Bar Charts for comparing Device Risk vs. IP Risk.

KPI Cards for "Total Transactions" and "Avg Device Risk" snapshots.
